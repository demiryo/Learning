Webmasters have a lot of roles taking care of a website. Such as designing the website, programming the website, marketing the website, and securing the website information. I would enjoy designing the website most, as that is the most creative role. As you can create a moving background or have the website glow where the mouse is hovering. You can also create cool animations as the user scrolls down.

Some responsibilities in the design role are to ensure the website is still readable. also to make the website still useable for it's purpose. also to make the website visualy good to look at so it is attractive and not jarring. for example, if the design role was done carelessly then the user may be confused by the layout. the user may also be annoyed if the website is visualy bland or boring, the user can also be annoyed if the website is visualy too crowded. it is the designer's job to take care of this. without the designer, the website will be innefective for it's job, no matter how well it is coded.
